Review
Viewport
여러 미디어쿼리 브레이크 포인트를 통해서 콘텐츠
및 레이아웃 구성이 변화하는 것을 Grid를 이용하여
대응하였다.
CSS transition
PC버전의 header 드롭다운 메뉴를 스크립트를
사용하지않고 CSS만을 이용해서 작동하게
만들어봤다.
Swiper.js
Grid와 Overflow를 통해 구현했던 슬라이드 효과를
간단한 플러그인 Swiper.js를 통해 편하게 구현하고
루프와 좌우 버튼을 통해 사용자 편의도 늘렸다.
Ajax & JSON
jQuery의 간편한 문법을 통해 Ajax를 사용해서
JSON의 객체 데이터를 불러왔고 Swiper.js가
적용된 콘텐츠를 채우는 데 활용하였다.
